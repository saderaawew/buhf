Для правильной работы компонентов, нам нужны изображения. 
Поскольку мы не можем автоматически создать изображения через API, 
можно использовать заполнители для имитации внешнего вида.

Вместо отсутствующих изображений можно использовать следующие пути:

cigar-cuban.png - заполнитель для кубинской сигары
cigar-nicaraguan.png - заполнитель для никарагуанской сигары
cigar-gold.png - заполнитель для золотой сигары 
hookah-egyptian.png - заполнитель для египетского кальяна
accessory-bowl.png - заполнитель для аксессуара (колба)
accessory-cutter.png - заполнитель для аксессуара (гильотина)
